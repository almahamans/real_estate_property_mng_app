# Real Estate Property Management App
## About the app:
It is an app to display list of properties with details, the user can manage the peoperties by adding and deleting properties.
## Technologies used:
- React js: to build the app.
- TailwindCss: to add style to the app.
- React Router: to navigate between pages.
- mockapi.io API: to build back-end.
- Fetch function: used to interact with api and display requests.
